# Todo List

Oddiy Todo List veb-ilovasi. React functional componentlar, Context API, JSON Server va Tailwind CSS ishlatilgan.

## Ishga tushirish

Paketlarni o'rnatish:

```bash
npm install
```

1-terminalda JSON Serverni ishga tushiring:

```bash
npm run server
```

2-terminalda React loyihani ishga tushiring:

```bash
npm run dev
```

Brauzerda Vite ko'rsatgan manzilni oching.

Ikkalasini bitta buyruq bilan ham ishga tushirish mumkin:

```bash
npm run start
```

## API

- `GET /todos` - vazifalarni olish
- `POST /todos` - yangi vazifa qo'shish
- `PATCH /todos/:id` - vazifa holatini o'zgartirish
- `DELETE /todos/:id` - vazifani o'chirish
